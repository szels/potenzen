# Potenzen Trainer

Eine interaktive Single-Page-App zum Trainieren von Quadratzahlen (x²) und dritten Potenzen (x³).

## Features

- 10 Multiple-Choice-Fragen pro Runde
- Zufällige Potenzen-Aufgaben (Quadratzahlen und dritte Potenzen)
- Highscore-System mit LocalStorage
- Responsives Design
- Sofortiges Feedback bei Antworten
- Fortschrittsanzeige

## Deployment auf GitHub Pages

1. Repository auf GitHub erstellen
2. Dateien hochladen
3. In den Repository-Einstellungen zu "Pages" gehen
4. Source: "Deploy from a branch" auswählen
5. Branch: "main" und Ordner: "/ (root)" auswählen
6. Speichern

Die App ist dann unter `https://[username].github.io/[repository-name]` erreichbar.

## Lokal testen

Öffne einfach die `index.html` Datei in einem Browser.

## Technologie

- HTML5
- CSS3 (mit Gradients und Animationen)
- Vanilla JavaScript
- LocalStorage für Persistenz
